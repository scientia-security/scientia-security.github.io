# [Scientia Security on GitHub](https://scientia-security.github.io/)
* 『[セキュリティコンサルタントの日誌から](https://www.scientia-security.org/)』の関連サイトです。
* このページでは、過去翻訳したドキュメント・執筆文書・プログラムを公開しています。
